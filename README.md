# Minify Buddy 📸

BASE用の画像を自動で最適化するWebツールです。

## 機能

- 画像を正方形（1:1）にトリミング
- 1280px × 1280pxにリサイズ（BASE推奨サイズ）
- 軽量なJPG形式で出力（画質85%）
- ドラッグ&ドロップで簡単操作
- 複数画像の一括処理に対応

## 使い方

1. ページにアクセス
2. 画像をドラッグ&ドロップ、またはクリックして選択
3. 自動で変換された画像をダウンロード
